# MCP Examples

HVN includes example MCP snippets for QA workflows. These files are not universal configuration files; adapt command names and paths to your agent client and installed MCP servers.

- `linear.example.json` shows the intended shape for Linear coordination when using an MCP-style integration.
- `ios-simulator.example.json` shows the intended shape for simulator QA.
- `browser.example.json` shows the intended shape for browser QA.

Use the examples with `hvn-linear-core`, `hvn-ios-sim-qa`, and `hvn-web-qa`. Keep credentials in your agent client's secret store.
