# ORCA-HVN Integrations

## What This Skill Is

A docs-first integration routing and setup skill for modern startup stacks across web, mobile, backend, auth, payments, analytics, automation, and business systems.

## Trigger

Use when the user asks which stack to choose, how to combine tools, how to set up an integration, or how to validate whether a stack component is configured correctly.

## Do Not Trigger

Do not use for random logo-list browsing with no workflow value.

## Required Inputs

- integration name, stack need, or target use case

## Optional Inputs

- web or mobile target
- active harness
- current stack choices

## Exact Workflow

1. Identify the product context: web, mobile, internal tool, SaaS, or AI product.
2. Route to the relevant stack guide or integration pack.
3. Surface platform-specific caveats, especially around auth, notifications, and payments.
4. Recommend the shortest useful paved-road bundle.
5. Hand off to setup or validation when appropriate.

## Expected Outputs

- stack recommendation
- integration pack reference
- setup or validation next step

## Quality Bar

Guidance should be opinionated, structured, platform-aware, and practical. Do not flatten web and mobile into the same advice.
