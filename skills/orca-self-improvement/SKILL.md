# ORCA Self Improvement

Use this skill when ORCA needs to reason explicitly about local adaptation versus framework evolution.

## Goal

Separate user-instance learning from framework learning, require evidence, and keep changes reviewable.

## Apply It To

- session-end review
- local preference adaptation
- framework issue candidates
- promotion from local to global
- improvement evaluation

## Rules

- local and global loops are separate
- weak signals should usually be watched, not promoted
- retries do not count as learning
- framework changes need stronger evidence than local changes

## Output

Produce:

- loop classification
- evidence summary
- proposed action
- review or approval need
