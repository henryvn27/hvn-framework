# Sample Intake

## Product

Meeting Brief Builder

## User

Consultants preparing for client calls.

## Job

Turn notes, account context, and prior action items into a concise meeting brief.

## Constraints

- Must work from pasted text in the first version.
- Must not claim CRM integration until implemented.
- Must make export or copy behavior obvious.

`orca-onboard` → `orca-spec` → `orca-plan` → `orca-build` → `orca-test-blind` → `orca-security` → `orca-ship`
