# Sample Spec

## Goal

Create a first version of Meeting Brief Builder that accepts pasted notes and returns a structured client meeting brief.

## Acceptance Criteria

- User can paste unstructured notes.
- User can generate a brief with agenda, risks, open questions, and follow-ups.
- User can edit generated sections.
- User can copy the final brief.
- Empty and error states explain recovery.

## Non-Goals

- CRM sync
- Calendar integration
- Multi-user collaboration
