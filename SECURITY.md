# Security Policy

## Supported Versions

ORCA Framework is currently in the `0.x` release line. Security fixes target the latest published release and the main branch.

## Reporting A Vulnerability

Report vulnerabilities privately to Henry Van Ness through the maintainer contact listed on the GitHub repository profile.

Include:

- A concise description of the issue
- Affected files or workflows
- Reproduction steps
- Expected impact
- Suggested mitigation if known

Do not open a public issue for a vulnerability until a maintainer confirms disclosure timing.

## Scope

Security reports may cover install scripts, command definitions, skill workflows, CI configuration, and documentation that could cause unsafe agent behavior.
