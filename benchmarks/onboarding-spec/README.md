# Onboarding And Spec Benchmark Pack

This pack helps compare how well ORCA Framework handles ambiguous intake and spec creation across versions or workflow changes.

## Contents

- `cases/` contains realistic benchmark cases
- [templates/benchmark-case.md](../../templates/benchmark-case.md) defines case structure
- [templates/benchmark-report.md](../../templates/benchmark-report.md) defines run reporting

## Recommended Use

1. run the same case set against the workflow versions being compared
2. keep the resulting onboarding and spec artifacts
3. score hard checks and rubric dimensions
4. compare both aggregate outcomes and failure patterns

## Starter Dimensions

- ambiguity detection
- question quality
- constraint capture
- edge-case awareness
- scope discipline
- acceptance criteria quality
- verification readiness
- overall spec usability
