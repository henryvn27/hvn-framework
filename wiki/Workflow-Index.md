# Workflow Index

HVN follows one shared lifecycle with host-specific adaptation.

## Main Stages

1. Route the work
2. Clarify the request
3. Discover constraints
4. Write the spec
5. Plan phases
6. Build
7. Review
8. QA
9. Ship
10. Preserve memory and follow-ups
