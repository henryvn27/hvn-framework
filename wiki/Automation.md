# Automation

ORCA automation exists to keep framework decisions current without turning reports into noise.

## Current Automation Jobs

| Job | Purpose |
| --- | --- |
| Ecosystem sweep | Watch Codex, Claude Code, GSD, wrapped packs, and related harness changes. |
| Compatibility reports | Track host support and fallback behavior. |
| Docs maintenance | Keep the public docs, wiki, and command references aligned. |
| Staleness checks | Flag old guidance before it becomes misleading. |
