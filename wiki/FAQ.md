# FAQ

## Is HVN only for Linear users?

No. Linear is the best-supported default path, but HVN can map the same artifacts and gates to another system of record.

## Do I need every doc before I start?

No. Start with `README.md`, `docs/start-here.md`, and `docs/quickstart.md`, then follow the path chooser.

## Is the wiki the source of truth?

No. The wiki is the summary layer. Canonical detail lives in `docs/`, `commands/`, `skills/`, `templates/`, and the install docs.
