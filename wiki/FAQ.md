# FAQ

## Do I Need Linear?

No. ORCA Framework is Linear-first, not Linear-only.

## Do I Need Every Feature?

No. Start with onboarding, spec, plan, build, and review. Add goal mode, background mode, subagents, and Orca Monitor only when they solve a real problem.

## Is The Wiki The Full Documentation?

No. The wiki is a routing layer. The Pages docs are the public manual, and `commands/` contains the command contracts.

## What Should I Read First?

`README.md`, then `docs/first-10-minutes.md`.

## Is ORCA Beginner-Friendly?

Yes, but it should not hide the real workflow. The goal is to make a new user competent faster, not trap them in simplified mode.

## Does ORCA Replace Codex Or Claude Code?

No. ORCA coordinates those hosts and uses native host features when they are available.
