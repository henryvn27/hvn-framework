# Core Concepts

Use this page when you need the main HVN ideas without reading every deep doc first.

## Core Terms

- Harness: the host runtime
- Work item: the durable record for meaningful work
- Spec: the execution contract
- Run memory: the compact continuation layer
- Blind QA: first-look testing without hidden context
- Runtime adaptation: host-specific behavior built on one shared workflow
