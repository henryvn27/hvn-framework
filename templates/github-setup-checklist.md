# GitHub Setup Checklist

## Need

- [ ] GitHub is required for the current workflow.
- [ ] GitHub is optional but useful.
- [ ] GitHub is not needed for this task.

## Desired Capabilities

- [ ] Read repository metadata.
- [ ] Read issues or pull requests.
- [ ] Create issues.
- [ ] Create branches or commits.
- [ ] Open or update pull requests.
- [ ] Read checks or workflow status.
- [ ] Create releases or changelog entries.

## Setup Path

- [ ] Native connector
- [ ] GitHub MCP server
- [ ] `gh` CLI
- [ ] API token
- [ ] Manual fallback

## Validation

- [ ] Auth status checked.
- [ ] Target repo reachable.
- [ ] Required read scope works.
- [ ] Required write scope works or is intentionally manual.
