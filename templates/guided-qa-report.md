# Guided QA Report

## Pass Type

State whether this is briefed QA or informed QA.

## Context Packet

List included context and withheld context.

## Target Flows

List flows, fixes, or acceptance criteria under test.

## Results

For each target, record pass, fail, blocked, or uncertain.

## Evidence

List screenshots, logs, accessibility notes, console output, simulator output, or manual observations.

## New Findings

List any issues discovered outside the target flow and whether they should block release.

## Recommendation

State next action.
