# Run Memory Contract

## Required Fields

- Memory type: fact, decision, blocker, preference, or assumption
- Scope: user, repo, project, or issue
- Statement
- Why it matters
- Source or evidence
- Freshness note

## Optional Fields

- Expiration condition
- Follow-up action

## Good Enough

- A future run can tell whether the memory is still likely useful.
- The entry does not masquerade as a trace.
