# Harness Installation

ORCA is harness-agnostic conceptually, but users still need concrete harness help.

## What A Harness Is

A harness is the agent surface or host where ORCA commands, skills, and workflow rules are actually used.

Examples:

- Codex
- Claude Code
- Hermes Agent
- VS Code

## Install Order

1. Install ORCA first.
2. Choose the harness in [harness-chooser.md](harness-chooser.md).
3. Follow host setup in [harness-setup.md](harness-setup.md).
4. Verify with [install-validation.md](install-validation.md).

## Core Rule

Do not force harness setup into the minimum ORCA install if the user is still deciding what host they want.
