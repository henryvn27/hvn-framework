# Plugins

Plugins are optional add-ons or setup surfaces that extend ORCA for specific workflows.

They are not part of the minimum install.

## Read This First

- [plugin-installation.md](plugin-installation.md)
- [plugin-troubleshooting.md](plugin-troubleshooting.md)

## Core Rule

Only install a plugin when:

- your next workflow actually needs it
- the value is clearer than the setup cost
- the harness or host can support it

## Plugin Buckets

- required core plugins: none by default
- optional helpful plugins: install only when your path needs them
- advanced or expert plugins: keep off the beginner path
