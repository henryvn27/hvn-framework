# Install Overview

ORCA installation should feel linear, not like a scavenger hunt.

Start here if you need the shortest explanation of the setup system before choosing a path.

## What You Are Installing

ORCA is the working surface.

The repository name is ORCA-HVN, but the install experience should answer one simple question:

> How do I get ORCA installed and working without guessing?

## The Three Install Paths

- Beginner path: [install-for-beginners.md](install-for-beginners.md)
- Standard path: [install.md](install.md)
- Technical path: [install-for-technical-users.md](install-for-technical-users.md)

## What Is Required

Core install requires:

- a supported operating system
- git
- a shell that can run the install scripts
- this repository

Some workflows also need:

- a harness such as Codex or Claude Code
- service connections such as GitHub or Linear
- optional plugins or integrations

Those are documented separately so new users do not have to read everything up front.

## Recommended Order

1. Read [install-prerequisites.md](install-prerequisites.md).
2. Choose a path in [install-paths.md](install-paths.md).
3. Follow the numbered guide in [install.md](install.md).
4. Run verification from [install-validation.md](install-validation.md).
5. Complete [first-run.md](first-run.md).
