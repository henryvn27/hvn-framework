# Graph Analysis Output Modes

ORCA Framework should choose the lightest useful graph or vault output first.

## Output Modes

### Quick Vault Summary

Use when the goal is simply to understand the major areas of the vault.

### Project Cluster Map

Use when notes appear to form multiple projects or domains and the user needs a clearer map.

### Orphan-Note Report

Use when navigation and hub coverage appear weak.

### Duplicate-Theme Report

Use when similar ideas or docs may exist under different names.

### Suggested Wiki Hub Structure

Use when graph or vault analysis reveals missing landing pages or duplicated routing.

### Graph Would Help Here

Use when basic inspection found signs that graph tooling could materially help, but the user has not asked for setup yet.

### Graph Not Needed

Use when the lighter path is sufficient and graph tooling would just add complexity.
