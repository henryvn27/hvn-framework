# Install Paths

Pick one path. Do not mix them unless you have a reason.

## Beginner Path

Use this if:

- you are new to ORCA
- you want the safest order
- you want more explanation

Go to [install-for-beginners.md](install-for-beginners.md).

## Standard Path

Use this if:

- you want the canonical setup flow
- you want step-by-step instructions without extra beginner framing

Go to [install.md](install.md).

## Technical Path

Use this if:

- you already use git and shell tools comfortably
- you want the shortest correct path

Go to [install-for-technical-users.md](install-for-technical-users.md).

## Decision Boxes

If you only want to try ORCA in one project:

- use local install

If you want ORCA available across projects:

- use global install

If you only need to read the framework or contribute to it:

- stay in repo mode
