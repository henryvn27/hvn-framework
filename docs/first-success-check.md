# First Success Check

Use this right after install.

## Success Sequence

1. run install verification
2. run doctor
3. read [first-workflow.md](first-workflow.md)
4. run `orca-onboard`
5. confirm you know the next command instead of guessing

## Validation Commands

```sh
./install/verify-install.sh --target ./.orca-hvn
./install/doctor.sh --target ./.orca-hvn
```

## Good First Success

A good first success is:

- one install target
- one verified setup
- one small ORCA task
- one clear next step
