# Harness Chooser

Pick the harness that matches how you actually work.

## If You Already Use A Specific Host

Use that host first. ORCA should adapt to it instead of forcing a fake neutral path.

## If You Do Not Know Yet

Start with the simplest host you can already access, then add more only if your workflow needs them.

## Decision Guide

If you want the current repo-native ORCA path:

- start with Codex

If you are already on Claude Code:

- use the Claude Code host guide

If you are running a controller/executor setup:

- read the Hermes and controller docs

If you need a generic fallback:

- use [hosts/generic.md](hosts/generic.md)
