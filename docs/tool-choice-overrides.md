# Tool Choice Overrides

ORCA Framework should be helpful, not evangelical.

## Default Override Behavior

If the user already picked a tool:

- honor that choice by default
- switch into setup or support mode
- explain caveats only when they materially affect success
- mention alternatives only when clearly useful

## Do Not

- nag the user into the default stack
- treat non-default as wrong by itself
- keep re-recommending tools the user already declined
