# Paved Road UX

Paved roads should hide unnecessary choice, not hide important platform differences.

## Integration Rule

A paved road should recommend:

- one sensible default bundle
- the required setup surfaces
- the validation path
- the safest fallback

It should not front-load optional integrations, comparison matrices, or long onboarding checklists when one path is enough.

## Examples

- web SaaS: Next.js + Vercel + Supabase + Stripe + Resend + PostHog + Sentry
- mobile app: Expo + React Native + Expo EAS + Supabase or Firebase + RevenueCat + OneSignal + Sentry
- internal tool: Next.js + Railway + Postgres + Clerk + GitHub + Linear
