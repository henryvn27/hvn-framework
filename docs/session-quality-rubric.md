# Session Quality Rubric

This rubric supports judgment. It does not replace judgment.

Use it when a session may have exposed framework-quality problems.

## Dimensions

- relevance
- clarity
- specificity
- adaptation to user context
- recovery after failure
- repetition versus novelty
- operational correctness
- trustworthiness

## Scale

Use a lightweight three-part scale:

- strong
- mixed
- weak

If a numeric score helps local comparison, use it only as a secondary shorthand.

## Usage Rule

The rubric should stay:

- short
- evidence-based
- explainable

Do not create the illusion of scientific precision.

The written evidence matters more than the score.
