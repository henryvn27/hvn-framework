# Plugin Installation

Use this page when a specific workflow needs a plugin.

## Plugin Setup Sequence

1. Confirm the plugin is actually needed.
2. Confirm your harness supports it.
3. Follow the plugin's authoritative install path.
4. Add ORCA-specific config only after the base plugin works.
5. Verify the plugin in isolation.
6. Verify the plugin inside the ORCA workflow that needs it.

## What Every Plugin Guide Should Say

- what the plugin does
- who should install it
- whether it is optional
- install steps
- config steps
- verification steps
- common errors
- disable or uninstall path
