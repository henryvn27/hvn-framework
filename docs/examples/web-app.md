# Web App Example

This example sequence fits a production web app feature.

1. `hvn-onboard` gathers user, task, success, and constraint context.
2. `hvn-discover` inspects routes, components, data flow, and test setup.
3. `hvn-spec` writes acceptance criteria for the user-facing flow.
4. `hvn-plan` breaks the change into implementation and verification phases.
5. `hvn-build` implements the feature.
6. `hvn-review` checks behavior and maintainability.
7. `hvn-test-blind` opens the app in a browser with no hidden context.
8. `hvn-test-briefed` retests fixes using a context packet.
9. `hvn-ship` prepares release notes and final checks.
