# Optional Tool Not Required

## Situation

The user asks for a local code change, and GitHub issue creation would be useful but not necessary.

## Guidance

GitHub is optional for this task. ORCA Framework can make and validate the local change now. If GitHub is connected, it can link the commit or prepare a PR. If not, it will leave a local summary and manual PR text.

## Done

The local change is complete when validation passes. GitHub follow-up is separate.
