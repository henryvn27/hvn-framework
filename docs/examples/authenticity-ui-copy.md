# Example: UI Plus Copy Calibration

## Task

Redesign an onboarding screen for a financial operations dashboard.

## Design Calibration

Avoid decorative gradients, generic dashboard cards, and abstract trust badges. Use compact operational layout, real task sequencing, clear account states, and restrained visual hierarchy.

## Writing Calibration

Audience: finance operators who want to know what is ready, blocked, and risky.

Avoid generic welcome copy. Use verbs tied to the workflow: reconcile, approve, investigate, export.

## Combined Decision

The screen should feel like a working operations tool, not a SaaS marketing page. Copy and layout should reduce uncertainty in the first thirty seconds.
