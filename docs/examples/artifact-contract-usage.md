# Example Artifact Contract Usage

## Scenario

A maintainer reviews a spec that sounds plausible but is missing non-goals and acceptance criteria.

## Contract Check

Using [templates/contracts/spec-contract.md](../../templates/contracts/spec-contract.md), the maintainer marks the spec incomplete because required fields are missing.

## Result

The spec is revised before planning starts. This prevents the plan from inventing scope and gives QA a real contract for done.
