# Example: Anti-Generic UI

## Task

Build a landing page for a debugging tool.

## Design Calibration

Avoid:

- Purple gradient hero
- Floating feature cards
- Icon circles
- Abstract "build faster" copy
- Decorative terminal mockups with fake logs

Direction:

- Lead with an actual debugging session.
- Use real log snippets, stack traces, and before-after evidence.
- Keep visual density moderate and technical.
- Make the primary action a concrete workflow: "Run a trace".

## Review Signal

If the page could be reused for any developer tool by swapping the logo, it failed authenticity preflight.
