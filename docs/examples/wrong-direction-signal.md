# Example: Wrong-Direction Signal

The user said to update the active ORCA Framework checkout, but ORCA Framework kept reasoning from an older local folder name until corrected.

Signal:

- wrong-direction signal
- poor failure-handling signal

Why it matters:

The issue was not just an isolated shell error. The workflow continued with the wrong framing until the path mismatch was explicitly corrected.

Outcome:

Candidate for runtime or orientation improvement if the pattern recurs.
