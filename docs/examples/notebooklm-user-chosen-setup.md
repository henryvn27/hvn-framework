# NotebookLM User-Chosen Setup

User signal:

> I already have a NotebookLM notebook. Help me connect it to this workflow.

Recommended ORCA Framework behavior:

- switch to setup-help mode
- do not argue for another knowledge tool unless there is a clear incompatibility
- identify whether the user is on the official enterprise path or a community path
- validate access, notebook reachability, and caveats
- connect the notebook to the current research, ideation, or documentation workflow
