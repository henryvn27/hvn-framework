# Aesthetic Profile Usage

## Product

Internal reconciliation dashboard.

## Selected Profile

## Effect On Calibration

Design calibration avoids marketing hero patterns, oversized cards, and decorative charts. It prioritizes tables, status rails, timestamps, and risk hierarchy.

Writing calibration uses operational verbs: reconcile, approve, investigate, export.

## Result
