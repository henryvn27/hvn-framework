# Manual Setup Fallback

## Situation

The host cannot use the needed service connector.

## Guidance

Continue with manual artifacts:

- paste the issue, PR, or task context
- run local verification
- write the spec, plan, QA report, or PR text locally
- ask the user to post the result in the external service

## Done

The phase is complete when the local artifact contains enough information for the user to update the service manually.
