# Example Pause And Resume

## Pause

A review run found unclear verification evidence. The coordinator paused the run and opened a checkpoint instead of declaring success.

## Inspect

The inspector artifact linked:

- trace summary
- metrics summary
- QA artifacts
- open blocker note

## Resume

After human review, the decision was `revise and resume`. The builder reran verification with clearer evidence and updated shared state before QA resumed.
