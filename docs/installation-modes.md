# Installation Modes

## Repo Mode

Repo mode uses this repository directly. It is best for contributors and teams that vendor ORCA Framework into a product repo.

## Local Mode

Local mode installs ORCA Framework into a project folder, usually `./.orca-framework`. It is best when a single product wants a stable copy of the framework.

## Global Mode

Global mode installs ORCA Framework into `$HOME/.orca-framework`. It is best when one user wants the same commands and skills available across projects.

## Choosing A Mode

Use repo mode while editing ORCA Framework itself. Use local mode for team projects that want committed framework artifacts. Use global mode for personal agent configuration.
