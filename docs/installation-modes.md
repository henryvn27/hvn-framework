# Installation Modes

## Repo Mode

Repo mode uses this repository directly. It is best for contributors and teams that vendor ORCA-HVN into a product repo.

## Local Mode

Local mode installs ORCA-HVN into a project folder, usually `./.orca-hvn`. It is best when a single product wants a stable copy of the framework.

## Global Mode

Global mode installs ORCA-HVN into `$HOME/.orca-hvn`. It is best when one user wants the same commands and skills available across projects.

## Choosing A Mode

Use repo mode while editing ORCA-HVN itself. Use local mode for team projects that want committed framework artifacts. Use global mode for personal agent configuration.
