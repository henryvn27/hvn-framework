# Harness Capability Matrix

This matrix summarizes how ORCA-HVN orchestration concepts map across harnesses.

## Capability Areas

- native subagents
- native side sessions
- structured delegation packets
- background worker support
- parent-worker observability
- approval surfaces
- result ingestion support
- fallback manual delegation support

## ORCA-HVN Priority

For orchestration-heavy work, ORCA should care most about:

- native subagents
- native side sessions
- background or detached execution
- branching or resume support
- independent review or checker surfaces

The fast summary should complement, not replace, deeper runtime and compatibility docs.
