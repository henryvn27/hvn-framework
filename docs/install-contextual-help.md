# Install Contextual Help

Install help should appear at the moment it is needed.

## Good Install Help Blocks

- What does this mean?
- Do I need this?
- Skip this if you are not using X.
- If this failed, try this.
- If this is your first time, use the beginner path.

## Rule

Do not front-load every concept before the user has taken the first step.

Use short callouts near the step where confusion usually appears.

But do ask a short setup interview early enough to avoid sending the user down the wrong install path.
