# Runtime Adaptation Guide

## When To Use This Guide

Use this guide when harness capability, setup state, or risk level changes what ORCA-HVN should recommend.

## What Runtime Adaptation Does

It chooses the safest reviewed path for the active environment instead of assuming feature parity.

## Questions It Answers

- does this harness support the feature well enough
- should the workflow fall back
- what setup path is recommended here
- which docs or guides apply

## Common Pitfalls

- copying a Codex path into a Claude context without checking
- recommending background or goal mode as defaults when capability is unclear
- missing integration setup caveats
