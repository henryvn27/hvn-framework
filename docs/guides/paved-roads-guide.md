# Paved Roads Guide

## When To Use Paved Roads

Use paved roads when you want the strongest recommended default path for a common workflow.

## What They Are

Paved roads are opinionated, optional workflow packs. They reduce decision fatigue without removing escape hatches.

## Best Use

- new feature delivery
- bugfix with QA follow-up
- legacy modernization slices
- long-running bounded execution

## Common Pitfalls

- treating every path as equally recommended
- staying on a paved road after the context no longer fits
- ignoring harness limits the road depends on
