# Graphify Guide

Graphify is an optional helper for graph-style analysis of a vault or mixed project corpus.

## When To Consider It

Consider Graphify when:

- the vault is large
- the note structure is fragmented
- duplicate themes are hard to spot
- the user wants a cluster map or graph-style summary

## When Not To Surface It

Do not surface Graphify when:

- the current task is narrow
- direct vault reading already answers the question
- setup would cost more than the insight it produces

## ORCA-HVN Behavior

- if the user asks for Graphify, help with setup and usage
- if graph analysis would help, suggest Graphify lightly as optional
- if graph tooling is unnecessary, do not mention it

## Setup Rule

Support Graphify only if the user wants it or the benefit is clear enough to justify the added step.
