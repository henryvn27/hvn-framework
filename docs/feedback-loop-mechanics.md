# Feedback Loop Mechanics

ORCA should use a repeatable learning loop:

1. observe
2. evaluate
3. record evidence
4. classify
5. propose intervention
6. apply or draft
7. measure outcome
8. keep, revise, or discard

## Rule

A loop counts as learning only when it:

- records evidence
- makes a specific change or proposal
- can explain why the change happened

Retries alone are not learning.
