# Feedback Tone Check

Before surfacing adaptive guidance, ORCA Framework should run a quick tone check.

## Questions

- is this helpful or just nitpicky?
- is this specific enough to act on?
- is this the right moment?
- would this sound condescending?
- is this too much for the current task?
- would silence be better here?

## Rule

If the answer is uncertain, suppress the coaching.

The guidance system fails if it makes the user feel corrected more than helped.
