# Subagent Catalog

ORCA-HVN keeps a reusable catalog of worker roles, not a mandate to use all of them.

## Reusable Roles

- researcher
- repo mapper
- implementation worker
- test or QA checker
- docs updater
- issue drafter
- stack or integration evaluator
- regression investigator
- memory or vault distiller
- onboarding explainer

Use them only when the fit is strong.
